# 5. Principles of Landscaping / लैंडस्केपिंग के सिद्धांत

| Principle | Meaning (Hindi) |
| :--- | :--- |
| **Balance** | संतुलन: Equal visual weight (समान भार) |
| **Proportion** | अनुपात: Correct sizing (सही आकार) |
| **Rhythm** | लय: Repetition/Movement (पुनरावृत्ति) |
| **Focal Point** | मुख्य बिंदु: Draws the eye (ध्यान आकर्षित करना) |
| **Harmony** | सामंजस्य: Uniform blending (एक साथ मिलना) |
